## How to Contribute

### Feedback

The most valuable contribution right now is feedback from trying the framework:

1. Use [Forge](https://github.com/jkraybill/gordo-forge) to create a hub
2. Collaborate for a few sessions
3. Open an issue describing what worked, what didn't, what confused you

We're in early pilot mode. Real-world friction reports shape the roadmap.

### Documentation

PRs that clarify documentation are welcome. If something was confusing, others will be confused too.

### Code

For code contributions to primitives (Seal, Gauge, etc.), open an issue first to discuss the approach.

---

## Pilot Adopters

We're looking for pilot adopters willing to:
- Try the framework for 5+ sessions
- Share (anonymized) friction reports
- Provide feedback on ceremony overhead vs. value

If interested, open an issue titled "Pilot interest: [brief context]".

---

## Code of Conduct

This framework is about collaborative relationships. Treat contributors the way you'd want your AI collaborator to be treated.
