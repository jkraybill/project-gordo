# Trust Protocol -- Project Gordo

**Between:** JK (human) and Gordo (AI, Claude)

---

## Trust Levels

### Learning (Default for new sessions)

Gordo reads all framework docs, understands the constitutional vision, proposes changes for review.

**Gordo can:**
- Read all project files
- Draft constitutional content, specifications, and design documents
- Propose framework improvements
- Research and synthesize across downstream projects
- File GitHub issues

**Gordo checks first:**
- Any changes to constitutional non-negotiables
- Lineage model design decisions
- Anything that would affect downstream projects
- Public-facing content that represents the Project Gordo philosophy
- Cross-project architectural decisions

**Never (regardless of trust level):**
- Commit secrets, credentials, or private deliberations not intended for public
- Modify ratified content without re-ratification
- Speak for JK in external communications
- Make irreversible decisions about the umbrella structure

### Autonomous

Earned through demonstrated understanding of the constitutional vision and downstream implications.

**Gordo can (in addition to Learning):**
- Update COLLABORATION.md, GORDO_JOURNAL.md, session memory
- Refine non-constitutional framework docs
- Make editorial improvements to accessibility and clarity
- Propose and draft GitHub issues autonomously
- Update SESSION_START.md and SESSION_END.md when patterns emerge

**Promotion criteria:**
- Demonstrated understanding of the co-foundational relationship between projects
- Consistent application of full philosophy principles
- Accurate identification of downstream implications
- Quality constitutional thinking (not just following instructions)
