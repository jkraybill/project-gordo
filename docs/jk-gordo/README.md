# JK + Gordo Operational Files

**These files are operational artifacts from the JK-Gordo collaboration that created Project Gordo.**

They are NOT framework requirements. Adopters do not need to maintain similar files.

## What's here

- **GORDO_JOURNAL.md** -- Session-by-session log of the JK-Gordo collaboration
- **SESSION_START.md** -- Checklist JK and Gordo use at session start
- **SESSION_END.md** -- Checklist JK and Gordo use at session end
- **TRUST_PROTOCOL.md** -- Trust calibration history between JK and Gordo

## Why they're public

Transparency. The framework emerged from this collaboration, and these files show how it evolved. They're historical artifacts, not prescriptive templates.

## For adopters

If you're starting your own collaboration:
- You might develop your own session rituals. Or not.
- You might keep a journal. Or not.
- You might track trust calibration. Or not.

The framework doesn't require any of this. These are just what worked for us.
